##VOICE CHANGER AI
## Libraries
 - Room
 - Retrofit
 - Dagger-Hilt
 - Navigation 
 - Coroutines
 - Glide
 - viewBinding

Voice Change is an application that allows a text written in Ai to be voiced with the voice of the selected person.
